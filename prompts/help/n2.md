# n2 — N² (N-squared) interface matrix

## SYNOPSIS
    syscribe -m <root> n2 [<qname>] [--depth <N>] [--format text|html|json]
                          [--interfaces-only] [--allocations]

## DESCRIPTION
Generates an N² interface matrix (§16) — a square chart with the in-scope `PartDef`/`Part`
elements on the diagonal; the off-diagonal cell at (row R, column C) lists the interfaces R
sends to C. Derived read-only from the model's existing `connections:`, `flowConnections:`,
and (with `--allocations`) `allocatedTo:` edges — no new authoring.

With no `<qname>`, every part in the model is on the axis. With a `<qname>`, the axis is the
composite element's subparts, expanded `--depth` levels (default 1).

## OPTIONS
    --depth N           Levels of nesting below <qname> to include (default 1).
    --format F          text (default) | html (table fragment) | json.
    --interfaces-only   Drop axis elements that have no interfaces.
    --allocations       Include allocation (allocatedTo) edges as well.
