---
type: Package
name: Ordering
---

Requirements for the generic `displayOrder` frontmatter field: a first-class numeric
ordinal, attachable to any element, that governs the order in which elements are
presented in ordered outputs (the requirements report, the coverage matrix, and the
web-UI containment tree). Motivated by requirement authoring, where the order in which
requirements are read carries meaning that identifier order alone does not capture.
