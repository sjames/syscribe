type: Package
name: HTML
---

Requirements for the `export-html` subcommand: a full-model, offline, multi-file static HTML
site with per-element pages, navigation, resolved cross-links, embedded diagrams, report pages,
and client-side search.

All requirements derive from `REQ-TRS-HTML-000` and are governed by `ADR-SYS-HTML-001`
(`Decisions::HTMLExportADR`).
