Analyses all failure pathways leading to unintended engine acceleration, as
defined by safety goal SG-ENG-001 (HE-ENG-001, ASIL D).

## Top event

demand for more than 100 ms.

## Methodology

Binary fault tree per IEC 61025. Basic event failure rates sourced from
IEC TR 62380 (automotive component database). Common-cause analysis performed
separately using the beta-factor method.
