name: Security

Security requirements for the Computer-Based Interlocking (CBI) system. Requirements in this package are derived from cybersecurity goals in the `Security/` TARA and are implemented by software components of the vital processor. Each requirement is traceable to at least one `CybersecurityGoal` via `derivedFromSecurityGoal:` and to the communication ADR via `breakdownAdr:`.
