Fault Tree Analysis package. Contains fault trees, logic gates, and basic events for quantitative safety analysis of the CBI system per IEC 61025. Each fault tree is rooted at a top event corresponding to a safety goal violation and targets the SIL 4 dangerous failure rate threshold of < 10⁻⁸ /h (IEC 61508 Table 2).
