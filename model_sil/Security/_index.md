Cybersecurity analysis package for the Computer-Based Interlocking (CBI) system. Contains Threat Analysis and Risk Assessment (TARA) sheets produced in accordance with ISO/SAE 21434 and the NIS2 Directive requirements for railway critical infrastructure. Security goals defined here are referenced from `Requirements/Security/` via `derivedFromSecurityGoal:` links.
