The software image instance deployed on the vital processor channels at Station 1 Interlocking. Contains all vital and non-vital software component instances configured for the Station 1 track layout (maxRoutes, conflictMatrixSize, signalCount, pointsCount, etc. are set to Station 1 values during commissioning).
