title: Requirement with both asilLevel and dalLevel
status: draft
reqDomain: software
verificationMethod: test

This element **shall** satisfy the test condition. Both `asilLevel:` (ISO 26262) and `dalLevel:` (DO-178C) are set on the same element — these standards are incompatible and should produce W703.
